# SQL Learning Journey

This repository contains a beginner-friendly SQL practice path with short, focused examples.

## Learning Path

1. `01_SELECT.sql` — retrieve rows and columns
2. `02_WHERE.sql` — filter rows with conditions
3. `03_DISTINCT.sql` — remove duplicate values
4. `04_LIKE.sql` — pattern matching
5. `05_BETWEEN.sql` — range filtering
6. `06_ORDER_BY.sql` — sort query results
7. `07_LIMIT.sql` — limit the number of rows returned
8. `08_GROUP_BY.sql` — aggregate data by group
9. `09_HAVING.sql` — filter grouped results
10. `10_JOIN.sql` — combine related tables
11. `11_INSERT.sql` — add new rows
12. `12_UPDATE.sql` — modify existing rows
13. `13_DELETE.sql` — remove rows

## How to Use

Run the `.sql` files against a database that has sample tables such as `students`, `courses`, and `enrollments`.

## Goal

Build a solid foundation in SQL for data analytics and AI/ML work.
